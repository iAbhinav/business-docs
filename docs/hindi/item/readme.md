# आइटम 
---

आप जिन प्रॉडक्ट्स का व्यापार करते हैं उन्हें आइटम पेज पर रेकर्ड कर सकते हैं।

* एक या अधिक आइटम की [इन्वॉस](../invoice/readme.md) बना कर आप अपनी [पार्टियों](../party/readme.md) को बेच सकते हैं 
* या फिर उसकी दो पार्टियों के बीच उसकी [दलाली](../deal/readme.md) कर एक डील बना सकते हैं । 

---

## नयी आइटम कैसे जोड़ें ?

1. एप में [आइटम](https://business.umun.in/#/home/item/Item/ecp) पेज पर जाएँ 
2. क्रीएट (<img src="https://unpkg.com/ionicons@4.5.10-0/dist/ionicons/svg/ios-create.svg" height="16px" width="16px">) पर क्लिक करें।
3. आइटम का नाम और दूसरी इन्फ़र्मेशन भरें।
4. सेव (<img src="https://unpkg.com/ionicons@4.5.10-0/dist/ionicons/svg/ios-checkmark.svg" height="16px" width="26px">) पर क्लिक करें।
